6. Scientific Calculator Functions

✔ Square Root

The Square Root function calculates the root value of a number instantly. It is useful in mathematics, engineering and science.

Example: √144 = 12

□ Square (x²)

The Square function multiplies a number by itself. It is commonly used in geometry, algebra and financial calculations.

Example: 15² = 225

xʸ Power

The Power function raises a number to a given exponent.

Example: 2⁵ = 32

1/x Reciprocal

The Reciprocal function calculates the inverse of a number.

Example: 1/4 = 0.25

π (Pi)

Pi (π) is a mathematical constant used in circle and geometry calculations.

Example: π ≈ 3.14159265

7. Memory Functions

  • MC – Clears all stored memory.
  • MR – Recalls the stored value.
  • M+ – Adds the current value to memory.
  • M− – Subtracts the current value from memory.
  • Ans – Recalls the previous answer.
