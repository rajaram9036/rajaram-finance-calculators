6. Scientific Calculator Functions

√ Square Root

The Square Root function calculates the root value of a number instantly. It is widely used in mathematics, engineering and science.

Example: √144 = 12

□ Square (x²)

The Square function multiplies a number by itself.

Example: 15² = 225

xʸ Power

The Power function raises a number to the specified exponent.

Example: 2⁵ = 32

1/x Reciprocal

The Reciprocal function returns the inverse value of a number.

Example: 1 ÷ 4 = 0.25

π (Pi)

Pi (π) is a mathematical constant used in circle and geometry calculations.

Example: π ≈ 3.14159265
