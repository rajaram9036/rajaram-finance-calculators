3. Basic Mathematical Operations

➕ Addition

Addition is used to add two or more numbers together. It is the most common mathematical operation used in daily life.

Example: 125 + 375 = 500


➖ Subtraction

Subtraction helps you find the difference between two values. It is useful for budgeting, shopping and accounting.

Example: 950 − 275 = 675


✖ Multiplication

Multiplication is repeated addition. It is widely used in business calculations, education and finance.

Example: 25 × 12 = 300


➗ Division

Division splits a number into equal parts. It is useful for averages, ratios and financial calculations.

Example: 500 ÷ 5 = 100


📊 Percentage

Percentage is used to calculate discounts, GST, tax, profit, loss, marks and interest.

Example: 20% of 500 = 100

6. Scientific Calculator Functions

√ Square Root

The Square Root function calculates the root value of a number instantly. It is useful in mathematics, engineering and science.

Example: √144 = 12


□ Square (x²)

The Square function multiplies a number by itself. It is commonly used in geometry, algebra and financial calculations.

Example: 15² = 225


xʸ Power

The Power function raises a number to a given exponent.

Example: 2⁵ = 32


1/x Reciprocal

The Reciprocal function calculates the inverse of a number.

Example: 1/4 = 0.25


π (Pi)

Pi (π) is a mathematical constant used in circle and geometry calculations.

Example: π ≈ 3.14159265


7. Memory Functions

  • MC – Clears all stored memory.
  • MR – Recalls the stored value.
  • M+ – Adds the current value to memory.
  • M- – Subtracts the current value from memory.
  • Ans – Recalls the previous answer.
